# .antigravity — Data Reliability Suite

This folder contains the operating instructions, agent roles, skills, prompts, and task plans for building a portfolio-grade Data Quality & ETL Validation Framework.

Project goal: build a Python-first framework that validates data pipelines, source-to-target consistency, schema integrity, data freshness, business rules, and regression quality using pytest, pandas, SQLAlchemy, PostgreSQL, Docker, and CI/CD.

Use this folder as the brain/context layer for Antigravity.
